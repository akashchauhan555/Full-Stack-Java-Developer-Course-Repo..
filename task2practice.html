list in HTML

HTML lists are used to create a group of related items and display them in the form of a list on the webpage.

Type of list:
  • Unordered List
  • Ordered List
  • Description List
  1. Unordered List:

    2. The unordered list items are marked with bullets. It is also known as bulleted lists.

    Exp.
    • Oreo Shake
    • Chocolate Shake
    • Mango Shake
    • Banana Shake
    • Kiwi Shake
    • Kitkat Shake

  2. Ordered List:

    3. In an ordered list, all list items are marked with numbers by default.

    Exp.
    1. Oreo Shake
    2. Chocolate Shake
    3. Mango Shake
    4. Banana Shake
    5. Kiwi Shake
    6. Kitkat Shake

  3. Description List:

    4. A description list is a list of terms, with a description of each term.

    Exp.
    Coffee
    -550mg
    Milk
    - 1 ltr Tetra Pack
